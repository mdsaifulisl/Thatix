## Project Description
Thatix is a responsive recipe website built using **HTML, CSS, JavaScript, and Bootstrap**.  
It allows users to search for their favorite recipes, view recipe details, and browse the latest recipes fetched from the [TheMealDB API](https://www.themealdb.com/api.php).

### Key Features
- Responsive layout for mobile, tablet, and desktop.
- Live search for recipes by name.
- Dynamic recipe cards with images, titles, and instructions.
- Modal view to show full recipe details.
- Scroll-to-top button for easy navigation.
- Footer with About section, quick links, and social media icons.

---

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)
- Bootstrap 5
- Axios (for API requests)
- Font Awesome (icons)
- TheMealDB API

---

## How to Use
1. Open the live URL or clone the repository.  
2. Type a recipe name in the search bar to filter recipes.  
3. Click "View Recipe" on any card to see full details in a modal.  
4. Use the scroll-to-top button to quickly navigate to the top.
