## Project Description
Thatix is a responsive recipe website built using **HTML, CSS, JavaScript, and Bootstrap**.  
The website allows users to:

- Search for their favorite recipes using the search bar.
- View a list of the latest recipes fetched from the [TheMealDB API](https://www.themealdb.com/api.php).
- Click on a recipe card to open a modal with full details including the image, title, and instructions.
- Scroll smoothly through recipes on mobile and desktop devices.

## Features
- **Responsive Design:** Works on mobile, tablet, and desktop screens.
- **Live Search:** Search recipes in real-time by name.
- **Dynamic Recipe Cards:** Recipes are dynamically rendered using JavaScript and API data.
- **Modal View:** Click on a recipe to see full details in a modal.
- **Scroll to Top Button:** Easily navigate to the top of the page.
- **Footer:** Contains About, Quick Links, and Social Media icons.

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)
- Bootstrap 5
- Axios (for API requests)
- Font Awesome (for icons)
- TheMealDB API

## How to Use
1. Open the live URL in a browser.  
2. Type a recipe name in the search bar to filter recipes.  
3. Click the "View Recipe" button on any card to see full details in a modal.  
4. Use the scroll-to-top button to quickly return to the top of the page.
